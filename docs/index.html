Rough thought in. Refined prompt out.

Promptify is a Chrome extension that rewrites vague, hasty prompts into clear, specific, high-quality ones — on any site, with your own API key.

Get Promptify Chrome Web Store listing coming soon · install from source today
Why Promptify

Small prompt, big difference.

Refine anywhere

Select text in ChatGPT, Claude, Gmail — right-click → Refine prompt, and it's rewritten in place.

Preview & undo

A side-by-side original vs refined view before anything changes, and one-click Undo after.

Your keys, your data

No servers, no accounts, no tracking. Requests go straight to OpenRouter or Anthropic with your key.

Profiles & site pins

"Coding" on GitHub, "Email" in Gmail — saved answer sets that pick themselves per site.

Three takes

×3 mode drafts three alternative refinements at different temperatures. Keep the winner.

Library & tweaks

Save your best prompts as snippets, insert them anywhere, and iterate with "make it shorter".

⌘ ⇧ U

One shortcut refines the selection in any input — no mouse required.
